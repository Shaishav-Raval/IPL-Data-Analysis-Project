# IPL-Data-Analysis-Project
End-to-end IPL Data Analysis project using SQL, Python, Machine Learning, and Power BI.

## Project Overview
This project analyzes Indian Premier League (IPL) match data to uncover insights about team performance, toss impact, and match outcomes. The analysis combines SQL, Python, and Power BI to perform data exploration, visualization, and machine learning prediction.

## Tools & Technologies Used
- Python (Pandas, Matplotlib, Scikit-learn)
- PostgreSQL
- Power BI
- Excel
- Git & GitHub

## Key Analysis Performed

### 1. Exploratory Data Analysis
- Team performance analysis
- Season-wise statistics
- Venue analysis
- Toss impact analysis

### 2. Data Visualization
Visualizations created using Python and Power BI to understand trends in match results and team performance.

### 3. Machine Learning Model
A Random Forest model was used to predict match outcomes based on features such as:
- Toss winner
- Toss decision
- Venue
- Competing teams

### Model Accuracy
The model achieved approximately **54% accuracy** in predicting match outcomes.

## Key Insights
- Certain teams show strong dominance in specific venues.
- Toss decision has measurable impact in certain conditions.
- Historical data can moderately predict match outcomes.
